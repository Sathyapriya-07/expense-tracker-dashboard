# Expense Tracker Dashboard

## Overview

Expense Tracker Dashboard is a web application that helps users manage their income and expenses efficiently.

## Features

* Add Income
* Add Expense
* View Balance
* Transaction History
* Local Storage Data Management

## Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage

## How to Run

1. Open the project folder in Visual Studio Code.
2. Open index.html using Live Server.
3. Add income and expense transactions.
4. View balance and transaction history.

## Conclusion

This project demonstrates financial tracking and management using front-end web technologies.
